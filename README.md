# Various debloater magisk modules for Poco devices  
 Debloat various system apps on Poco devices.  
 Works on both MIUI 13 and MIUI 12. Should work other MIUI based devices too.   
 Removes these non-removable apps from system without breaking anything:  
## Stage 1:  
(just installing [stage 1 module](https://github.com/symbuzzer/Poco-Debloater-Magisk-Modules/tree/main/stage%201) module will remove all)  
  > -Analytics  
  > -Facebook App Manager  
  > -Joyose  
  > -MIUI Daemon  
  > -MSA Global
  > -GetApps  
  > -Payment Service  
  > -Xiaomi SIM Activation Service  
  > -MiPay  
  > -Facebook Installer  
  > -Facebook Services  
  > -Xiaomi App Vault  
  > -Yellow Page  
  > -Google One  
  > -MiPay Service  
  > -Soter Service  
  > -IFAA Service
